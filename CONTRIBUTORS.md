## How to Contribute

1. Fork the repo
2. Create your branch: `git checkout -b feat/your-feature`
3. Commit your changes: `git commit -m 'feat: add something great'`
4. Push to your branch: `git push origin feat/your-feature`
5. Open a Pull Request against `main`

Please read the [Contributing](README.md#-contributing) section in the README for code style guidelines.
